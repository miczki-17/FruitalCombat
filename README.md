🍎 Fruit: Frutal Combat<br>
Zabawna, dynamiczna gra akcji/survival napisana w C++ i SFML. Wybierz swojego owocowego bohatera i odeprzyj nieskończone fale inteligentnych, zmutowanych warzyw!<br><br>

📖 Krótki zarys fabularny (Hook)<br>
W sercu Wielkiej Dżungli od tysiącleci panował pokój. Jednak gdy Mroczny Nawóz skaził Ziemię, zwykłe warzywa zyskały świadomość i... mordercze intencje. Jako jeden z ostatnich owocowych wojowników, musisz przetrwać i dotrzeć do "jądra skażenia".<br><br>

✨ Główne mechaniki<br>
4 grywalne postacie (Jabłko, Banan, Pomarańcze, Cytryna) o unikalnych statystykach.<br>
Rozbudowany sklep z ulepszeniami permanentnymi (Meta-progression).<br>
Dynamiczny system fal z potężnymi Bossami.<br>
Zbieranie monet, power-upów i odblokowywanie osiągnięć.<br>
🛠️ Technologie<br>
C++17<br>
SFML 3.x<br>
Visual Studio 2022<br>
