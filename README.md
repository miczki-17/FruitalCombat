🍎 Fruit: Frutal Combat<br>
Zabawna, dynamiczna gra akcji/survival napisana w C++ i SFML. Wybierz swojego owocowego bohatera i odeprzyj nieskończone fale inteligentnych, zmutowanych warzyw!

📖 Krótki zarys fabularny (Hook)
W sercu Wielkiej Dżungli od tysiącleci panował pokój. Jednak gdy Mroczny Nawóz skaził Ziemię, zwykłe warzywa zyskały świadomość i... mordercze intencje. Jako jeden z ostatnich owocowych wojowników, musisz przetrwać i dotrzeć do "jądra skażenia".

✨ Główne mechaniki
4 grywalne postacie (Jabłko, Banan, Arbuz, Cytryna) o unikalnych statystykach.
Rozbudowany sklep z ulepszeniami permanentnymi (Meta-progression).
Dynamiczny system fal z potężnymi Bossami.
Zbieranie monet, power-upów i odblokowywanie osiągnięć.
🛠️ Technologie
C++17
SFML 3.x
Visual Studio 2022
