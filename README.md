🍎 Fruital Combat
Zabawna, dynamiczna gra akcji/survival napisana w C++ i SFML3. Wybierz swojego owocowego bohatera i odeprzyj nieskończone fale inteligentnych, zmutowanych warzyw!

📖 Krótki zarys fabularny (Hook)
W sercu Wielkiej Dżungli od tysiącleci panował pokój. Jednak gdy Mroczny Nawóz skaził Ziemię, zwykłe warzywa zyskały świadomość i... mordercze intencje. Jako jeden z ostatnich owocowych wojowników, musisz przetrwać i dotrzeć do "jądra skażenia".

✨ Główne mechaniki
4 grywalne postacie (Apple, Cherry, Orange, Strawberry) o unikalnych statystykach.
Rozbudowany sklep z ulepszeniami w trakcie gry (roguelike)
Dynamiczny system fal z potężnymi Bossami.
Zbieranie biomasy, power-upów i odblokowywanie osiągnięć.
🛠️ Technologie
C++17
SFML 3.x
Visual Studio 2022
